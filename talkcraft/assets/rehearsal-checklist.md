# Rehearsal Checklist

- Promise spoken clearly in first `2` minutes
- Each fence named out loud
- Each fence timed with a clock
- Every major claim backed by one concrete example
- No slide carries more than one core idea
- One board, demo, or visual move tested before delivery
- Transitions spoken, not implied
- Close stronger than final content slide
- Q&A starts after a finished ending
- One trim candidate ready if time slips
- One recovery move ready if demo fails
- One likely objection prepared for each core fence
